1NF (First Normal Form):
Atomic values, removing repeating groups.

2NF (Second Normal Form):
Eliminating partial dependencies by separating tables.

3NF (Third Normal Form):
Removing transitive dependencies (non-key dependencies).

BCNF (Boyce-Codd Normal Form):
Ensuring every determinant is a candidate key.

4NF (Fourth Normal Form):
Removing multi-valued dependencies (e.g., students with multiple hobbies).
